## Brand Guidelines

### Color Palette

**Primary Colors:**
- **Stellar Blue:** #7B61FF (Trust, Technology)
- **Ajo Gold:** #FFB800 (Prosperity, Community)
- **Deep Purple:** #4A148C (Wisdom, Heritage)

**Secondary Colors:**
- **Success Green:** #00C853 (Growth, Achievement)
- **Warm Orange:** #FF6F00 (Energy, Warmth)
- **Neutral Gray:** #424242 (Balance, Professionalism)

**Background Colors:**
- **White:** #FFFFFF
- **Light Gray:** #F5F5F5
- **Dark:** #1A1A1A

### Typography

**Primary Font:** Inter (Modern, Clean)
- Headlines: Inter Bold (700)
- Body: Inter Regular (400)
- Captions: Inter Medium (500)

**Secondary Font:** Poppins (Friendly, Approachable)
- Use for emphasis and callouts

### Logo Usage
- Minimum size: 120px width
- Clear space: 20px on all sides
- Always use on contrasting background
- Never distort or rotate

## Accessibility Checklist

For every graphic:
- [ ] Color contrast ratio meets WCAG AA standards (4.5:1 for text)
- [ ] Text is readable at small sizes
- [ ] Important information not conveyed by color alone
- [ ] Alt text written for screen readers
- [ ] Animations can be paused
- [ ] Flashing content avoided (seizure risk)
