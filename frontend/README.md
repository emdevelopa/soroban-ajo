# Frontend (Coming Soon)

This directory is reserved for the future web interface of Soroban Ajo.

## Planned Features

- **Group Dashboard**: Visual interface to create and manage savings groups
- **Member Portal**: Track contributions, upcoming payouts, and group status
- **Wallet Integration**: Connect with Freighter and other Stellar wallets
- **Real-time Updates**: Live updates on group activities and payments
- **Mobile Responsive**: Optimized for mobile and desktop browsers

## Technology Stack (Proposed)

- **Framework**: React/Next.js
- **Stellar Integration**: @stellar/stellar-sdk, @stellar/freighter-api
- **Styling**: Tailwind CSS
- **State Management**: React Query for contract state
- **Testing**: Jest + React Testing Library

## Development Timeline

Frontend development is planned for **Phase 3** of the roadmap (see [roadmap.md](../docs/roadmap.md)).
